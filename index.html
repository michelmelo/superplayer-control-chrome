Superplayer Control

Uma extenção para controle do Superplayer.fm

Download da extenção

Atalhos

Parar/Continuar Música 

Windows/Linux: Ctrl + Shift + 1
Mac: Command + Shift + 1

Pular Música 

Windows/Linux: Ctrl + Shift + 2
Mac: Command + Shift + 2

Amar Música (curtir) 

Windows/Linux: Ctrl + Shift + 3
Mac: Command + Shift + 9

Odiar Música (odiar) 

Windows/Linux: Ctrl + Shift + 4
Mac: Command + Shift + 0
